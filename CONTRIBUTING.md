# Contributing to AI Model Lineage
## Introduction
Contributions are welcome and encouraged. This document outlines the guidelines for contributing to the AI Model Lineage project.
## PR Workflow
1. Fork the repository
2. Create a new branch
3. Make changes and commit
4. Submit a pull request
## Commit Standards
* Use clear and concise commit messages
* Follow standard professional guidelines
## Code Style Rules
* Use PEP 8 guidelines for Python code
* Use Markdown for documentation
